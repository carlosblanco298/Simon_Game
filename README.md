# Simon Game

 ```

Se realizó una aplicación web que contiene un juego de memoria. Las tecnologías utilizadas fueron:
 - HTML5 para la estructura de la página
 - CSS para dar estilo de la estructura
 - JavaScript para dar funcionalidad y comportamiento a la aplicación web

 ```

## Ejecución

```
Esta aplicación web se ejecuta con Open Live Server, Live Preview, entre otras.
```
